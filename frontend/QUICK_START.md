## üìç Route Structure

### Public Routes
- `/landing` - Modern landing page with hero, features, testimonials
- `/auth` - Authentication page (login/register)
- `/onboarding` - First-time user onboarding (if needed)

### Protected Routes (require authentication)
- `/` - Home dashboard
- `/chat` - Support chat
- `/resources` - Resource directory
- `/safety-plan` - Safety plan builder
- `/evidence` - Evidence vault
- `/education` - Education hub
- `/settings` - Settings
